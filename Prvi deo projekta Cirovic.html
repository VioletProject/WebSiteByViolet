Google Chrome is a web browser that was first used on 3 September 2008 and the process of the publication of Google Chrome has gone through several stages. The comic of Scott McCloud was used by bloggers to present the functionality of the primary version of Google's browser. The changes that have much effect on the development of this browser were starting to occur in that period. All of these changes have governed direction of the earlier version of Google's browser. Blogger Philipp Lenssen made a scanned copy of the 38-page comic that was available on his website after receiving it on 1 September 2008. The comic was appointed on Google Books. This new browser was allegedly named "Chrome" because Google wanted to minimize the chrome of the browser. There are a number of versions of Google Chrome browser, each of these versions was performing in accordance with the circumstances.
